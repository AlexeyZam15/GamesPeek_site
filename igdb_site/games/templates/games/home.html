About GamesPeek

GamesPeek is a comprehensive video game discovery platform powered by the IGDB API. Our database contains thousands of games across all major platforms including PC, PlayStation, Xbox, Nintendo Switch, and mobile devices.

Whether you're looking for similar games to your favorites, want to compare game features side by side, or simply browse the latest releases - GamesPeek provides powerful search and filtering tools to help you find your next gaming adventure.

Key Features

  • Search games by title, genres, platforms, themes, and keywords
  • Intelligent similar game algorithm based on multiple criteria
  • Compare game specifications, ratings, and features
  • Filter by release year, rating, popularity, and many more parameters
  • Detailed information including system requirements, developers, publishers
