Complete Video Game Database - Find Games Similar to Your Favorites

GamesPeek is a comprehensive video game database that helps you find games similar to the ones you already love. With a database of over 45,000 games, our intelligent similarity algorithm analyzes multiple criteria to recommend titles you'll enjoy.

Unlike traditional game stores that only show popular titles, our game database lets you search by genres, keywords, themes, platforms, release years, developers, game modes, and engines. Then our algorithm finds games that share these characteristics.

How Our Game Database Similarity Algorithm Works

  • Genre matching (30% weight) - Action, RPG, Strategy, Adventure, Indie, Shooter, Simulation
  • Keyword analysis (25% weight) - Gameplay mechanics, features, content tags
  • Theme similarity (20% weight) - Fantasy, Sci-fi, Horror, Historical, Open world, Survival
  • Developer relationships (10% weight) - Same developers often create similar games
  • Player perspective (8% weight) - First-person, Third-person, Top-down, Isometric
  • Game modes (7% weight) - Single player, Multiplayer, Co-op, Couch co-op, Split screen

Frequently Asked Questions About Our Game Database

Is GamesPeek's game database free to use?

Yes! Our complete video game database is completely free. No registration required to search, browse, or compare games.

How many games are in the database?

Currently over 45,000 games across all major platforms including PC, PlayStation, Xbox, Nintendo Switch, and mobile.

Can I compare two games in the database side by side?

Absolutely! Click "Compare" on any game card to see ratings, release dates, platforms, and features compared from our database.

Where does the game database data come from?

GamesPeek uses the official IGDB (Internet Game Database) API, the same database that powers major gaming platforms.
