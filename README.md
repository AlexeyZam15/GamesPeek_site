# Games Peek

Django-приложение для загрузки данных об играх из IGDB API, их хранения в базе данных и поиска похожих игр.

## 🎯 Функциональность

### Основные возможности:
- **Загрузка данных из IGDB** - автоматическая выгрузка информации об играх через IGDB API
- **Поиск похожих игр** - интеллектуальный подбор схожих игр на основе различных критериев
- **Управление базой данных** - администрирование и обновление данных об играх
- **REST API** - программный интерфейс для доступа к данным
